# Deployment

## Showcase Deployment

The Primer showcase (demo site) is automatically deployed:

- **Frontend**: Automatically deployed to Vercel
  - Preview deployments are created for each pull request
  - Production deployment happens when PRs are merged to `main`
  - Production URL: [https://primer-frontend.vercel.app](https://primer-frontend.vercel.app)
  - See [Vercel Deployment Guide](./vercel-deployment.md) for details

- **Backend**: Automatically deployed to Strapi Cloud
  - Production deployment happens when PRs are merged to `main`
  - See [Strapi Cloud Deployment Guide](./backend/development/strapi-cloud-deployment.md) for details
