# @primershop/strapi-plugin-product-actions

A Strapi plugin that enables product actions functionality.

## Installation

```bash
npm install @primershop/strapi-plugin-product-actions
```

## Configuration

Add the plugin to your Strapi configuration file (`config/plugins.ts`):

```typescript
export default () => ({
  "primershop-product-actions": {
    enabled: true,
  },
});
```

## Features

- Product actions management
- Series product actions

## Requirements

- Strapi >= 5.22.0
